AULA #1 leia o conteúdo abaixo !

🟢 Aula 01 – Introdução ao JavaScript


Nesta aula, pequeno gafanhoto 🐛, você conhece JavaScript, a linguagem que dá vida aos sites. Enquanto o HTML cria a estrutura e o CSS cuida do visual, o JavaScript é responsável por fazer o site pensar e reagir.

Aprendemos que JavaScript roda diretamente no navegador e permite criar interações em tempo real, como responder a cliques, digitação e movimentos do mouse.

O pequeno gafanhoto entende que programar é resolver problemas usando lógica, escrevendo comandos claros para o computador.


🧪 Variáveis – Guardando informações

Variáveis servem para guardar dados na memória, como nomes, números e resultados. 

let nome = "Arthur"
let idade = 15

Aqui o computador guarda os valores e pode usá-los depois no programa.


🧠 Condições – Tomando decisões

Com if e else, o programa aprende a tomar decisões sozinho. 

if (idade >= 18) {
  console.log("Maior de idade")
} else {
  console.log("Menor de idade")
}

O pequeno gafanhoto aprende que o código segue caminhos diferentes dependendo da condição.


🔁 Repetições – Fazendo o código trabalhar

Laços de repetição permitem executar comandos várias vezes sem escrever tudo de novo. 

for (let i = 1; i <= 5; i++) {
  console.log(i)
}

Isso deixa o código mais curto, inteligente e eficiente.


Ao final dessa aula, o pequeno gafanhoto entende que:

  • JavaScript controla o comportamento do site
  • Variáveis guardam informações
  • Condições tomam decisões
  • Repetições automatizam tarefas

🟢 Aula – JavaScript Completo para Iniciantes


Nesta aula mais avançada, pequeno gafanhoto 🐛, você aprende JavaScript de forma completa, desde os conceitos básicos até ideias usadas em sites e sistemas reais.

O JavaScript permite criar páginas inteligentes, que mudam sem precisar recarregar.


📦 Tipos de dados

Aprendemos que o JavaScript trabalha com diferentes tipos de dados. 

let nome = "Maria"
let idade = 20
let aprovado = true

Cada tipo serve para uma finalidade e ajuda o programa a funcionar corretamente.


🧠 Funções – Organizando o código

Funções servem para organizar e reutilizar código, evitando repetição. 

function somar(a, b) {
  return a + b
}

Assim, o pequeno gafanhoto escreve menos código e resolve mais problemas.


📚 Arrays – Listas de dados

Arrays permitem guardar vários valores em uma variável. 

let frutas = ["Maçã", "Banana", "Uva"]

Isso é muito usado em sistemas, jogos e aplicações reais.


🖥️ DOM – Controlando o HTML

Uma das partes mais importantes: manipular o HTML com JavaScript. 

document.getElementById("titulo").innerHTML = "Novo Texto"

Aqui o JavaScript passa a mandar na página, mudando textos, cores e estilos.


🖱️ Eventos – Reagindo ao usuário

Eventos permitem que o site reaja às ações do usuário. 

botao.addEventListener("click", clicar)

Isso torna o site interativo e dinâmico.


Ao final dessa aula, o pequeno gafanhoto:

  • Domina a base do JavaScript
  • Entende lógica de programação
  • Consegue criar sites interativos
  • Está pronto para projetos maiores
