Aulas Descritas


AULA #1-5 leia o conteúdo abaixo !

📘 Aulas 01 a 05 – Fundamentos do Python (com exemplos)


No começo do curso aprendemos o que é programação. Programar significa dar ordens claras ao computador.

👉 Exemplo de programação na vida real:
Quando você pede para alguém fazer um sanduíche e diz: "pegue o pão, coloque o queijo e feche". Se você não explicar direito, a pessoa pode errar. O computador funciona do mesmo jeito.

Na Aula 01, aprendemos que Python é uma linguagem fácil de aprender e ideal para iniciantes.

Na Aula 02, vimos para que o Python serve. Ele pode ser usado para jogos, sites e automações.

👉 Exemplo:
Em um jogo, o Python pode decidir: se o jogador perdeu toda a vida, o jogo acaba.

Na Aula 03, aprendemos a instalar o Python e conhecemos o IDLE.

👉 Representação:
O Python é o motor e o IDLE é o painel de controle. Sem o motor, nada funciona.

Na Aula 04, escrevemos nossos primeiros comandos. O mais importante foi o print().

👉 Exemplo de código:
print("Olá, pequeno gafanhoto!")
Esse comando faz o computador mostrar uma mensagem na tela.

Na Aula 05, aprendemos que o Python lê o código linha por linha, de cima para baixo.

👉 Exemplo de erro comum:
Se você tentar usar algo antes de explicar ao Python, o programa não vai entender.

Exemplo de ordem correta:

print("Mensagem 1")
print("Mensagem 2")

O Python executa exatamente nessa ordem. Se trocar as linhas, o resultado muda.

Conclusão:
Das aulas 01 a 05, aprendemos como pensar como programador, como escrever comandos simples e como o Python entende nosso código.

AULA #2 leia o conteúdo abaixo !

📘 Aulas 06 a 11 – Tipos, Cálculos, Textos e Decisões


A partir da Aula 06, começamos a entender que o Python trabalha com diferentes tipos de dados. Nem toda informação é igual.

👉 Exemplo:
Um número como 10 não é igual a um texto como "10". Para o Python, isso faz muita diferença.

Na Aula 06 aprendemos os tipos primitivos: int, float, str e bool.

👉 Exemplo de representação:
idade = 15
nome = "Arthur"
aprovado = True

Na Aula 07 usamos o Python como uma calculadora inteligente. Aprendemos os operadores aritméticos.

👉 Exemplo de cálculo:

soma = 5 + 3
multiplicacao = 4 * 2

O Python respeita a ordem de precedência, assim como na matemática da escola.

Na Aula 08 aprendemos a usar módulos, que são códigos prontos que facilitam a vida.

👉 Exemplo:
Se quisermos calcular uma raiz quadrada, não precisamos criar tudo do zero.

import math
print(math.sqrt(16))

Na Aula 09 aprendemos a trabalhar com textos (strings). Textos também podem ser analisados e modificados.

👉 Exemplo:
Podemos contar letras, mudar para maiúsculo ou pegar partes de uma palavra.

nome = "Python"
print(nome.upper())

Na Aula 10, o Python começou a tomar decisões. Usamos if e else.

👉 Exemplo de decisão:

nota = 7
if nota >= 6:
  print("Aprovado")
else:
  print("Reprovado")

Na Aula 11, aprendemos que o programa pode ter mais de uma condição, usando o elif.

👉 Exemplo com várias decisões:

nota = 5
if nota >= 7:
  print("Excelente")
elif nota >= 6:
  print("Aprovado")
else:
  print("Reprovado")

Conclusão:
Das aulas 06 a 11, aprendemos a trabalhar com dados, fazer cálculos, manipular textos e criar programas que pensam e decidem.

Esses conteúdos transformam programas simples em programas inteligentes.
